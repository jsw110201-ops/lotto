# 🎉 로또 번호 추첨기 + 폭죽 + 랜덤 응원

## 소개
이 프로젝트는 **모바일 친화적인 로또 번호 추첨기**입니다.  
화면을 터치하면 6개의 로또 번호가 자동으로 생성되고, **폭죽 애니메이션**과 함께 **100개의 랜덤 응원 문구** 중 하나가 표시됩니다.  

로또 추첨과 시각적 즐거움, 응원 메시지를 동시에 경험할 수 있는 웹 프로젝트입니다.

---

## 주요 기능

- **모바일 터치 지원**
  - 화면을 터치하면 로또 번호 추첨 시작
  - PC에서도 클릭으로 사용 가능

- **로또 번호 추첨**
  - 1~45 범위, 중복 없는 랜덤 6개 번호
  - 번호는 오름차순 정렬
  - 둥근 공 형태로 시각화

- **폭죽 애니메이션**
  - 번호 추첨 시 Canvas 기반 폭죽 효과 발생
  - 랜덤 색상과 움직임으로 화려한 시각 효과

- **랜덤 응원 문구**
  - 100개의 문구 중 하나 랜덤 선택
  - 번호 추첨 후 화면에 표시

---

## 사용 방법

1. HTML 파일을 다운로드합니다.
2. 웹 브라우저 또는 모바일에서 파일을 엽니다.
3. 화면을 터치(또는 클릭)하면 번호가 자동으로 추첨되고, 폭죽과 응원 메시지가 표시됩니다.

---

## 기술 스택

- **HTML5**: 페이지 구조 및 터치 이벤트
- **CSS3**: 디자인 및 번호 시각화
- **JavaScript (ES6)**: 로또 번호 생성, 랜덤 메시지, 폭죽 애니메이션
- **Canvas API**: 폭죽 시각 효과

---

## 화면 구성

- **상단**: "인생 여전 말고 인생 역전" 문구
- **중앙**: 추첨된 로또 번호
- **하단**: 랜덤 응원 문구
- **배경**: 터치 시 폭죽 애니메이션
